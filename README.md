# Gerenciador de produtos

Gerenciador de produtos e matérias-primas utilizando **React**, **Redux Toolkit**, e **Tailwind CSS**.

## Tecnologias

- React Router 7
- Redux Toolkit
- Tailwind CSS
- TypeScript
- Vite

## Configuração

1. Instalação de dependências:
```bash
npm install
```

2. Copiar o .env.example para .env e substituir a URL da API:
```
VITE_API_URL=http://localhost:8080
```

3. Rodar o servidor de desenvolvimento:
```bash
npm run dev
```

4. Abra [http://localhost:5173](http://localhost:5173)
